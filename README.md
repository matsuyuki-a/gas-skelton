gas-skelton
============

[Google Apps Script](https://script.google.com/home) を TypeScript で書くためのスケルトンコードです. [clasp](https://github.com/google/clasp) 使用.

## 使用方法
- https://script.google.com/home/usersettings からAPIからのApps Scriptの使用を許可しておく
- `npm install`
- `npm run create`
- `npm start`
